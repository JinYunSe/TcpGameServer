# TcpGameServer

### 필수과제

- [x] 프로젝트 생성 및 이벤트 별 코드 처리
- [x] 환경변수 및 상수, 에러 처리
- [x] 프로토콜 버퍼 적용 및 패킷 파싱
- [x] 유저 세션 및 게임 세션 관리
- [x] 접속 및 이동 패킷 교환
